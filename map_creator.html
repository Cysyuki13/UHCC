🖱️ Select tool: click to select, drag or use arrow keys / buttons to move.
Other tools: click to add new object. Right‑click to delete. Ctrl+Z / Ctrl+Y undo/redo.
📥 Import: paste JSON (from export) or load file to restore a map.
🖐️ Pan: Space + drag or middle‑click + drag or Shift + left drag.
🎯 Axis lock while moving: hold Shift while dragging or using arrow keys.
⭐ Spawn points: up to 6, will be used as player starting positions.
🛡️ Safe Zone: define areas where players cannot place blocks during the draft phase.
🎥 Camera bounds: define the rectangle where the camera is allowed to pan (players can still move beyond).
⬇️ Void Threshold: red dashed line shows where players fall into the void.
